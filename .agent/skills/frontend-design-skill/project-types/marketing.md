# Project Type: Marketing

Use for landing pages, homepages, campaign pages, product marketing, waitlists, and conversion-first brand surfaces.

## Priority order

1. Clear value proposition
2. Strong primary CTA
3. Trust and credibility
4. Narrative pacing
5. Distinctive brand expression

## Research focus

Study:
- how comparable products communicate value above the fold
- proof patterns: testimonials, logos, metrics, demos, case studies
- objection handling
- CTA placement and repetition
- scroll pacing and section rhythm

## Questions to resolve

- Who is the target audience?
- What is the offer?
- What action should the user take?
- What objections are likely?
- Is the tone more trust-first, premium, playful, or high-energy?

## Common failure modes

- pretty hero with vague copy
- too many equal CTAs
- features without user outcomes
- decorative sections with no conversion role
- generic “future of X” startup language

## Good outcomes

A strong marketing page makes the offer understandable quickly, builds belief, and moves the user toward one primary action.
